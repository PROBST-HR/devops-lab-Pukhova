# Lab1 — Docker Basics Setup

## Цель работы

Целью данной лабораторной работы является изучение основ контейнеризации с использованием Docker. Студент должен научиться создавать образы, запускать контейнеры и управлять ими.

---

## Задачи

В рамках лабораторной работы необходимо выполнить следующие задачи:

- Установить Docker Desktop на Mac  
- Изучить базовые команды Docker  
- Запустить тестовый контейнер `hello-world`  
- Работать с готовыми образами (Ubuntu, nginx)  
- Создать Dockerfile и Flask приложение  
- Настроить контейнер с пробросом портов и проверкой работы приложения  
- Изучить работу с томами (volumes)
